About this book

Who is this book for?

This book is meant as a tool to help people who are still getting used to code want to move beyond transcribing tutorials, and think of drawing with code as they would think of drawig with a pencil. This book is also meant for people who have experience with code, and would prefer a momment of levity and freedom to draw as they will.

How was this book made?

The examples in this book use p5.js, a creative coding framework based in javaScript. The code editing was made possible by CodeMirror, a tool for simulating a text editor in a browser.
